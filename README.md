# My NestPro

## Description
This is a project to learn NestJS.

## Installation

```bash
$ npm install
```

or you can `clone` the project and run `npm install` in the project directory.
```angular2html
$ git clone https://github.com/ivandi1980/my-nestpro.git
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
